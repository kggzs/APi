# 📦 APi 接口合集

> 一个实用的 PHP 工具集合，包含抖音视频解析和访问信息收集等功能

---

## 📚 项目导航

### 🎬 [抖音无水印视频解析](./dy.md)
- **文件**: `dy.php`
- **功能**: 解析抖音分享链接，获取无水印视频下载地址
- **特性**: 支持API调用和Web界面，安全防护完善

### 🎭 [访问信息收集脚本](./shell.md)
- **文件**: `shell.php`  
- **功能**: 收集访问者详细信息并显示警告页面
- **特性**: IP定位、VPN检测、设备信息采集

---

## 🚀 快速开始

### 抖音视频解析

```bash
# Web 界面使用
访问: http://your-domain.com/dy.php

# API 调用
curl "http://your-domain.com/dy.php?api=1&url=抖音链接"
```

### 访问信息收集

```bash
# 直接访问即可触发记录
访问: http://your-domain.com/shell.php
```
